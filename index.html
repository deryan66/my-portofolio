Specialized In

Microsoft Office

Microsoft Office adalah kumpulan aplikasi produktivitas yang terdiri dari Word (pengolah kata), Excel (pengolah data/spreadsheet), PowerPoint (presentasi), dan Outlook (email & kalender). Aplikasi ini membantu pengguna dalam mengerjakan tugas kantor, sekolah, atau keperluan pribadi untuk membuat dokumen, menghitung data, dan berkomunikasi.

WEB DEVELOPMENT

Web development adalah pembuatan website yang terdiri dari Front-End (tampilan yang dilihat user - HTML, CSS, JavaScript) dan Back-End (server dan database - PHP, Python, Node.js). Keduanya bekerja sama membentuk website yang fungsional. Pengembang yang menguasai keduanya disebut Full-Stack Developer.

Design Canva

Canva adalah platform desain online yang memungkinkan pengguna membuat desain visual dengan mudah melalui template siap pakai. Fitur utamanya meliputi editor drag-and-drop untuk membuat presentasi, poster, media sosial, logo, dan undangan dengan berbagai elemen desain seperti font, gambar, dan ilustrasi yang bisa disesuaikan tanpa perlu keahlian desain profesional.

PHP, HTML, CSS

PHP, HTML, dan CSS adalah tiga bahasa dasar pembuatan website. HTML berperan sebagai struktur website (seperti teks, gambar, tombol), CSS mengatur tampilan dan tata letaknya (warna, ukuran, posisi), sedangkan PHP menangani logika dan pemrosesan data di server (form, database, login). Ketiga bahasa ini bekerja sama membentuk website yang fungsional dan menarik.

Vidio Editor

Editor video adalah software untuk membuat dan mengedit video. Fitur utamanya meliputi pemotongan klip, transisi antar scene, penambahan musik/suara, efek visual, teks/subtitle, dan koreksi warna. Software populer termasuk Adobe Premiere Pro (profesional), DaVinci Resolve (gratis & pro), Vegas Pro, Filmora (pemula), dan CapCut (mobile). Editor dapat mengubah rekaman mentah menjadi video yang menarik untuk konten sosial media, film, atau keperluan bisnis.
